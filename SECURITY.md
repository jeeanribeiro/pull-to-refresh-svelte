# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅        |

## Reporting a vulnerability

Please report vulnerabilities privately via
[GitHub security advisories](https://github.com/jeeanribeiro/pull-to-refresh-svelte/security/advisories/new)
rather than opening a public issue.

You can expect an acknowledgement within a week. Once a fix is available it
ships as a patch release with credit in the release notes (unless you prefer
to stay anonymous).
